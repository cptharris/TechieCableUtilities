TechieCableUtilities

TCUManual

TechieCableUtilities is a lightweight, independent program made by TechieCable. It includes some of TechieCable's favorite utilities - all in one small program! TCU currently includes:

  • AlwaysOnTop - pin programs "on top" of all other programs
  • AOTMenu - use the AOT feature in a small context menu
  • TouchPad - turn touchpad on/off
  • SpecChars - type Unicode characters (or any phrase) with ease and speed

TCU comes with some simple options for its users.

The version of this manual available on Github may contain more recent instructions.

Note: stars * in the parentheses () represent the level of complexity, with five [*****] being most difficult and one [*] being the least difficult. The parentheses () also note what the instructions apply to.

Open TechieCableUtilities' taskbar menu

Many of these items require you to open the TechieCableUtilities taskbar menu

  • Find TechieCableUtilities in your taskbar
    • Generally in the lower right corner
    • Click the small upward-facing arrow
    • Look for the TechieCableUtilities icon
  • Right-click the TechieCableUtilities icon

View the TCUManual in-app (General *)

  • Open the taskbar menu
  • Click "Open TCUManual"

OR

  • Find TechieCableUtilities in your taskbar
  • Double-click TechieCableUtilities

Turn features on/off (General *)

  • Open the taskbar menu
  • Enter the "Options (Hotkeys)" menu
  • Left-click features on/off

**This feature disables/enables the HOTKEYS (mentioned below)

Pin/Unpin a program "on top" (AlwaysOnTop **)

  • Pin a program
    • Select the program you want to put "on top"
    • Press CTRL + ALT + SHIFT + Space
  • Unpin a program
    • Select a pinned program
    • Press CTRL + ALT + SHIFT + SPACE

To activate/deactivate the touchpad (TouchPad *)

  • Press CTRL + F2

OR

  • Open the taskbar menu
  • Left-click "TouchPadToggle"

** TouchPad only works on some computers and requires a touchpad to be included in the computer. Please check if your computer has a touchpad before using

** TouchPad relies on an external script (app\TouchpadToggle.exe) to work. You may have to allow this program through your security/antivirus program

Use the AlwaysOnTop Menu (AlwaysOnTop **)

  • Press CTRL + ALT + SHIFT + UP
  • Select a program from the list
  • Do the same to unpin a program

** You may have to activate the program (click it) for it to recognize that it is no longer "on top"

Disable a feature (General ***)

  • Add the following to TCU.ini

[disabled]

  • And then choose one or more of the following:

disable_AOT=1
disable_AOTMenu=1
disable_TouchPad=1
disable_loading=1

  • Remove any of the above to allow the feature again (or set =0 instead)

** Use disable_loading if you don't want to see the TCULauncher loading screen

Move TechieCableUtilities

  • Reinstall TechieCableUtilities with TCUSetup.exe, but specify an alternate location
  • When the installation process has finished (but before clicking "Finished"!), move your TCU.ini and \usr\ from the old installation location
  • Delete the old installation
  • Click "Finish" to run TechieCableUtilities from the new location

Top Menu

The top menu is the menu under the TechieCableUtilities item at the top of the toolbar menu. Access it by opening the taskbar menu and hovering over "TechieCableUtilities".

Visit TechieCable's website (General *)

  • Click "website" under the TechieCableUtilities menu

Backup your settings/configs (General *)

  • Click "backup"
  • First time backing up?
    • Choose the default backup location
  • TechieCableUtilities will create a backup in the (pre)selected folder

Delete TechieCableUtilities (General *)

If you don't want TechieCableUtilities anymore, something is broken, or you want to uninstall to reinstall, you can remove TechieCableUtilities from your computer. To do this, you can either delete TechieCableUtilities' folder or use the built-in removal function

  • OPTION ONE:
    • Open the taskbar menu and click "Remove TCU" (under the "TechieCableUtilities" menu)
    • TechieCableUtilities will ask you if you would like to backup your settings
    • A small window may briefly appear; this is the command deleting the folder
    • Done!
  • OPTION TWO:
    • Open the taskbar menu and click "Open Script Directory"
    • Open the taskbar menu again and click "Exit"
    • In the window that opened, go up a level and delete the folder called "TechieCableUtilities".

Customization

Use a custom icon for the options menu (General ***)

  • Replace app\settings_cog.ico with your own .ico file
  • Reload the program

** To revert to the original, simply delete app\settings_cog.ico and restart the program

Use custom hotkeys: Part 1 (General **)

In v1.0.8, customizing hotkeys became so much easier!:

  • Open the taskbar menu and click "Hotkey Customizer"
  • In each box, type the hotkey you want to use
  • When you finish, click "Submit"

** Some keys may not work - if you really want to use one of these, follow the directions below

Use custom hotkeys - Part 2 (General ****)

  • These instructions are more advanced - refer to Part 1 unless you have to use this method
  • Add the following to TCU.ini (unless it is already present)

[hotkey]

  • And then specify the keyboard shortcut after the following

AOT_key=_____
AOTMenu_key=_____
TouchPad_key=_____

  • Visit the AutoHotKey docs website for instructions on writing hotkeys

Change the default backup location (General ***)

  • Remove the defaultBackupFolder entry from TCU.ini
  • Backup TechieCableUtilities

Add Special Characters (SpecChars *****)

  • Create a file in the \app\ directory called SpecChars.txt
  • Add :*:, followed by the characters that are replaced
  • Then add :: followed by the Unicode ID for the character you want to appear in {}

For example, :*:()::{U+00B0} replaces () with {U+00B0}, which is °

  • Try placing :*:()::{U+00B0} in your SpecChars.txt file to test it out!

HINT: Try searching for "Unicode " followed by the character you want.

Addons (General *****)

This is a very advanced feature. Make sure to read the resources found at the AutoHotKey docs website

  • TechieCableUtilities searches its usr directory for a file called addon.txt and runs anything in that file
  • If you need to add a callable function, put it in a file called gosub.txt (in the usr directory) and call it with "gosub" or a timer in the addon.txt file
  • If you are feeling really adventurous, try adding code into this file (in AutoHotKey)!
  • TechieCableUtilities is written in AutoHotKey, which you can learn more about at www.autohotkey.com
